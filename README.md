# A1 Sniper Signal Bot

This repository contains the A1 Forex/Gold Sniper Signal Bot.

The bot runs:
- XAUUSD (1m entry)
- GBPUSD (15m entry)
- EURUSD (15m entry)

SMC rules: BOS, CHoCH, FVG, OB, Premium/Discount, Volume, RSI, MACD, EMA.

This repository will be deployed to Render for 24/7 hosting.
